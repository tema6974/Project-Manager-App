# Project-Manager-App
Приложение для менеджмента проектов с функционалом добавления, изменения, удаления новых проектов, задач для них, сотрудников и клиентов. Хранение данных в БД на локальном сервере  в MySQL Workbench.



Проект предоставляет функционал для:
•	Управления информацией о клиентах, включая добавление, редактирование и удаление записей.
•	Учета сотрудников, с возможностью их назначения на проекты и задачи.
•	Ведения информации о проектах, включая сроки, статус и состав участников.
•	Управления задачами, с указанием их статуса и ответственных сотрудников.
• Поиск по названию записей в каждой вкладке.
+ еще другие фичи.
Разработанная система позволяет автоматизировать учет данных, снижает количество ручной работы и обеспечивает удобство в анализе и обработке информации. Таким образом, программа является эффективным инструментом для управления проектами, задачами и рабочими процессами, способствуя повышению продуктивности и прозрачности работы.
